**Show Games**
----
Accéder à toutes les parties

* **URL**

  /games

* **Method:**

  `GET`

**Show Game**
----
Accéder à une partie via son id

* **URL**

  /games/:id

* **Method:**

  `GET`

*  **URL Params**

   **Required:**

   `id=[integer]`
